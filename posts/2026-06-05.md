Everyone argues about Claude vs GPT. Meanwhile the actual work happens in open-source repos most marketing teams have never opened.

The model is the engine. These repos are the parts that make it do real work. Here's what I actually run, by task:

→ Orchestrate a team of agents — CrewAI. Role-based agents that hand work to each other: one researches the prospect, one drafts, one reviews. This is how content and prospecting pipelines get built — not one giant prompt.

→ Turn the live web into data — Firecrawl. Point it at any site and get back clean, model-ready text. Prospect research, competitive intel, lead lists — the work you used to pay someone to copy-paste.

→ Do things that have no API — browser-use. An agent that drives a real browser: clicks, fills forms, navigates. For the half of the web that never exposed an API.

→ Wire your stack to any model — MCP servers. The standard plug. Official ones already exist for HubSpot, Meta Ads, GA4, Webflow — your CRM and ad accounts become tools the agent can call directly.

→ Run deep research on autopilot — open-deep-research. Give it a question and it loops: search, read, refine, repeat, until it hands you a sourced brief.

None of these are secret. They're free, MIT-licensed, most with tens of thousands of GitHub stars.

The repos are commodity. The moat is the system you wire them into — and knowing which one to reach for.

That's the part nobody can copy from a star count.
